## LiveKit Voice Agents LLM Benchmarking

Benchmarking the LLM models available for free in the LiveKit Voice Agents SDK.  

[![View Notebook](https://img.shields.io/badge/View-Analytics%20Notebook-orange?logo=Jupyter)](analytics.ipynb)
[![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python)](https://www.python.org/)

## Repository Contents

- **[analytics notebook](analytics.ipynb)** — Results of LLM benchmarking in LiveKit Voice Agents.  
- **[llm_metrics.py](llm_metrics.py)** — Code for gathering LLM performance metrics.  
- **[tts_metrics.py](tts_metrics.py)** — Similar metrics gathering for the TTS stage.  
- **[logs](logs)** — Measurement results from different models.

Instructions for running LiveKit Voice Agents using the `uv` package manager are available in the [LiveKit Voice AI quickstart](https://docs.livekit.io/agents/start/voice-ai-quickstart/).
